ChangeAlert Library Documentation

ChangeAlert is a lightweight, customizable JavaScript alert notification library designed to enhance your web applications by providing visually appealing, interactive, and easy-to-use alert messages. It supports multiple alert types, sound notifications, custom icons, themes, and callback functions to create a dynamic user experience.

Library Features

  • Simple and highly customizable alerts.
  • Supports 4 alert types: Success, Error, Warning, and Info.
  • Multiple customizable properties including position, theme, sound, timeout, etc.
  • Callback function support (onClick, onClose).
  • Responsive design optimized for mobile devices.

How to Use the Library

1. Include the CSS and JS files

Include the CSS link in the <head> of your HTML, and place the JS script tag just before the closing </body> tag for best performance. 

<link rel="stylesheet" href="https://cdn.jsdelivr.net/gh/rajkumarnimod/changealert/dist/style.css" />
<script src="https://cdn.jsdelivr.net/gh/rajkumarnimod/changealert/dist/script.js"></script>
Copied!

2. Using ChangeAlert

Now you can easily use the ChangeAlert functions in your JavaScript to display alerts: 

ChangeAlert.success("Operation completed!");
ChangeAlert.error("An error occurred!");
ChangeAlert.warning("This is a warning!");
ChangeAlert.info("Just an informational message!");
Copied!

You can also pass optional properties to further customize the alerts: 

ChangeAlert.success("Success!", { position: "bottom-left", timeout: 5000 });
ChangeAlert.error("Error!", { sound: false, theme: "dark" });
Copied!

Properties

  • 1. message (Required)

    Type: string

    Description: The text content displayed inside the alert box.

    ChangeAlert.success("Data uploaded successfully!");
    Copied!
  • 2. type (Optional)

    Type: string | Default: "info"

    Description: Defines the alert type. Available options:

    • "success": For successful operations.
    • "error": For error messages.
    • "warning": For warnings.
    • "info": For informational messages. 
    ChangeAlert.error("Failed to load data!");
    Copied!
  • 3. position (Optional)

    Type: string | Default: "top-right"

    Description: Position of the alert on the screen. Options include:

    • "top-left"
    • "top-right"
    • "bottom-left"
    • "bottom-right"
    ChangeAlert.success("Task completed!", { position: "bottom-left" });
    Copied!
  • 4. timeout (Optional)

    Type: number | Default: 3000 (3 seconds)

    Description: Duration (in milliseconds) before the alert automatically closes. 

    ChangeAlert.success("Data saved successfully!", { timeout: 5000 });
    Copied!
  • 5. icon (Optional)

    Type: string | Default: Depends on alert type (e.g., "✅" for success)

    Description: Icon displayed inside the alert box. Can be an emoji or custom SVG. Defaults to a predefined icon based on the alert type. 

    ChangeAlert.success("Successfully completed!", { icon: "🎉" });
    Copied!
  • 6. closeButton (Optional)

    Type: boolean | Default: true

    Description: Show or hide a close (X) button to allow users to manually dismiss the alert. 

    ChangeAlert.error("Something went wrong!", { closeButton: false });
    Copied!
  • 7. animation (Optional)

    Type: string | Default: "fade"

    Description: Animation style for showing and hiding alerts. Options include:

    • "fade"
    • "slide"

    Custom CSS animations can also be applied.

    ChangeAlert.warning("Please check the details", { animation: "slide" });
    Copied!
  • 8. theme (Optional)

    Type: string | Default: "light"

    Description: Theme of the alert box. Available themes:

    • "light" (default)
    • "dark"
    ChangeAlert.success("Operation completed!", { theme: "dark" });
    Copied!
  • 9. sound (Optional)

    Type: boolean | string | Default: undefined (no sound)

    Description: Sound played when the alert is shown.

    • true: Play default sound based on alert type (success, error, warning, info).
    • false: Mute sound.
    • string: URL of a custom sound file.
    ChangeAlert.success("Data uploaded successfully!", { sound: true });
    Copied!
    ChangeAlert.success("File uploaded!", { sound: "https://example.com/success.mp3" });
    Copied!
  • 10. pauseOnHover (Optional)

    Type: boolean | Default: true

    Description: Whether to pause the alert's auto-close timer when the user hovers over it. 

    ChangeAlert.warning("Alert with timer pause on hover!", { pauseOnHover: false });
    Copied!
  • 11. onClick (Optional)

    Type: function | Default: null

    Description: Callback function executed when the alert box is clicked. 

    ChangeAlert.success("Click to acknowledge!", {
          onClick: () => alert("Alert clicked!")
        });
    Copied!
  • 12. onClose (Optional)

    Type: function | Default: null

    Description: Callback function executed when the alert closes (manually or automatically). 

    ChangeAlert.error("Something went wrong!", {
          onClose: () => console.log("Alert closed")
        });
    Copied!

Combining Multiple Properties

You can effortlessly combine various properties when calling the ChangeAlert function. For example, you can set the type, position, timeout, sound, theme, and a custom icon all in one go! 

ChangeAlert.success("File uploaded successfully!", {
  type: "success",           // Alert type
  position: "top-left",      // Position on screen
  timeout: 5000,             // Alert will disappear after 5 seconds
  sound: true,               // Play default sound for success
  theme: "dark",             // Dark theme
  icon: "🎉",                // Custom icon (emoji)
  closeButton: true,         // Show close button
  pauseOnHover: true,        // Pause the timer on hover
  onClick: () => alert("You clicked the alert!"), // Callback on click
  onClose: () => console.log("Alert closed!") // Callback on close
});
